# LÖVE 2D Development Setup Guide

## Installation Steps (Windows)

1. **Download LÖVE 2D**
   [https://love2d.org/#download](https://love2d.org/#download)

2. **Add LÖVE 2D to your Windows Path**
   * Press the **Windows Key** and search for "Edit the system environment variables". Open it.
   * Click the **Environment Variables...** button.
   * In the bottom list ("System variables"), look for the variable named `Path` and select it, then click **Edit**.
   * Click **New** and paste the path to the folder where Love2D is installed:
       `C:\Program Files\LOVE\`
   * Click **OK** on all windows.

3. **Restart VS Code completely.**

4. **Download VS Code**
   [https://code.visualstudio.com/download](https://code.visualstudio.com/download)

5. **Install Extensions**
   * **Lua Language Server:** [Marketplace Link](https://marketplace.visualstudio.com/items?itemName=sumneko.lua)
   * **Debugging:** [Marketplace Link](https://marketplace.visualstudio.com/items?itemName=tomblind.local-lua-debugger-vscode)
   * **Formatting Guide:** [Getting Started Guide](https://maxzsol.com/getting-started-with-love2d-and-vs-code-a-beginners-guide/)

6. **Install Git**
   [https://git-scm.com/install/](https://git-scm.com/install/)

7. **Create Shortcut**
   Navigate to the LOVE folder in Program files (`C:\Program Files\LOVE\`). Drag and drop the `love.exe` file onto your desktop to make a shortcut.

8. **Optional Step**
   Rename the shortcut to say **"LOVE"**.

9. **Setup Project Folder**
   Create a folder called **"LOVE_Code"** on your desktop to hold the LOVE 2D project folders.
   *(Note: To run your game, the specific project folder inside here will be dragged and dropped onto the LOVE shortcut icon).*

10. **Get the Project**
    Clone the GitHub Repository or download the project folder into the `LOVE_Code` folder.
